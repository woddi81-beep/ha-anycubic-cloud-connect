# Anycubic Cloud Connect

Home Assistant custom integration for Anycubic 3D printers through Anycubic Cloud.

Benutzerdefinierte Home-Assistant-Integration fuer Anycubic-3D-Drucker ueber die Anycubic Cloud.

Integration personnalisee Home Assistant pour imprimantes 3D Anycubic via Anycubic Cloud.

## Highlights

- Cloud polling through the `anycubic-cloud-mcp` PyPI package.
- UI config flow.
- Sensors, binary sensors, buttons, switches, and services.
- English, German and French README files.
- UI translations for English, German, French and Russian.

---

- Cloud-Abfrage ueber `anycubic-cloud-mcp`.
- Einrichtung ueber die Home-Assistant-Oberflaeche.
- Sensoren, Binary-Sensoren, Buttons, Schalter und Services.

---

- Interrogation cloud via `anycubic-cloud-mcp`.
- Configuration depuis l'interface Home Assistant.
- Capteurs, capteurs binaires, boutons, interrupteurs et services.

## Notes

This is an unofficial community integration. It depends on Anycubic Cloud APIs and may require updates if Anycubic changes its cloud behavior.

Dies ist eine inoffizielle Community-Integration und haengt von Anycubic-Cloud-APIs ab.

Il s'agit d'une integration communautaire non officielle qui depend des API Anycubic Cloud.
